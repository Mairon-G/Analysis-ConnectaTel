# 📊 ConnectaTel Customer Behavior Analysis

## Overview

This project explores customer behavior data from **ConnectaTel**, a telecommunications company. The analysis focuses on **data cleaning**, **exploratory data analysis (EDA)**, **customer segmentation**, and generating business insights from customer usage patterns.

## Objectives

- Assess and improve **data quality**
- Analyze customer demographics and service usage
- Identify outliers and usage trends
- Segment customers by age and consumption behavior
- Provide actionable business recommendations

## Tools Used

- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Jupyter Notebook**

## Analysis Highlights

### Data Cleaning

- Handled missing values and invalid records
- Corrected anomalous age values
- Identified inconsistent date entries

### Exploratory Data Analysis

The analysis examined:

- Age distribution
- Call frequency and duration
- Messaging activity
- Plan adoption patterns

### Customer Segmentation

Customers were grouped according to:

- **Age**: Young, Adult, Senior
- **Usage Level**: Low, Medium, High

### Key Findings

- Most customers are **low-to-moderate service users**.
- Usage distributions are **right-skewed**, with a small group of heavy users.
- Plan type has limited influence on customer behavior.
- High-usage customers present opportunities for **premium offerings** and targeted marketing.

## Business Recommendations

- Develop specialized plans for heavy users.
- Implement behavior-based marketing campaigns.
- Improve customer retention through segmentation.
- Strengthen data validation and quality controls.
